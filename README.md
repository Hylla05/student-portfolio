# Student Portfolio and Academic Management Website

## About the Project

My name is Hyllamanda Reuben, and I am a 100-level Software Engineering student at Miva Open University.

I developed this website as an academic project to present my educational background, skills, interests, sample projects, and academic activities. The project also demonstrates my understanding of HTML, CSS, and JavaScript.

## Website Features

The website contains five main pages:

- **Home:** Introduction, photograph, biography, and welcome audio
- **About:** Educational background, career aspiration, technical skills, and hobbies
- **Projects:** Three sample software project concepts
- **Academic Planner:** An interactive task manager for adding, completing, filtering, and deleting tasks
- **Contact:** A contact form with JavaScript validation

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Flexbox
- CSS Grid
- Responsive web design
- DOM manipulation
- JavaScript form validation

## Academic Planner

The academic planner allows users to:

- Add academic tasks
- Assign a course, due date, and priority
- Mark tasks as completed
- Filter tasks
- Delete tasks
- View task statistics and completion progress

The tasks are temporarily stored in a JavaScript array and reset when the page is refreshed.

## Contact Form Validation

JavaScript is used to ensure that:

- No field is left empty
- The email address is in a valid format
- The phone number contains digits only
