# Hyllamanda Reuben — Student Portfolio

Android-friendly flat upload version. All website files are in this one directory so they can be selected individually in GitHub's mobile upload screen.

## Important

Upload the extracted files, not the ZIP file and not the containing folder. `index.html` must appear directly on the repository's main page.

## Files

- Five HTML pages
- `styles.css` — external responsive stylesheet
- `script.js` — planner, navigation, animations, and validation
- Four WebP images and one SVG favicon
- `welcome.mp3` — multimedia introduction

## Publish

In the GitHub repository, open **Settings → Pages**. Choose **Deploy from a branch**, select **main**, select **/ (root)**, and save.
